# watchWyy - 腕上云音乐
*腕上云音乐* 是一个能在 Apple Watch 
上独立收听网易云音乐歌曲的软件。无需依赖手机。

# 功能简述
- [x] 播放来自网易云音乐的歌曲
- [x] 支持用户登录
- [x] 播放远程歌单
- [x] 搜索歌曲、歌单
- [x] 查看评论、喜欢音乐
- [x] 更多功能等您探索……
